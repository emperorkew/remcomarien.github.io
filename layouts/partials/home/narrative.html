Process description

From pharmaceutical chemistry to code — a non-linear career path that ended up making sense.

01

API synthesis

Pharmaceutical engineering — reaction chemistry, crystallization, GMP manufacturing. Understanding how to take a molecule from lab scale to commercial production while keeping regulators happy and yields high.

02

Process automation

Paper batch records, manual data entry, Excel-based trending — pharma manufacturing was stuck in 2005. I started writing code to fix it. DCS interfaces, automated reporting, electronic batch records. The factory floor meets the terminal.

03

Continuous improvement

Now I work at the intersection: PAT (Process Analytical Technology), real-time process monitoring, predictive models for CQAs, and the software that ties it all together. Making pharma manufacturing smarter, not just compliant.
